# BuildBetter MCP setup

Use this MCP to pull meeting insights and call summaries from BuildBetter.

## What you need
- A BuildBetter account
- An API key

## Steps
1. Sign in to your BuildBetter dashboard.
2. Create or copy an API key.
3. Set `BUILDBETTER_API_KEY` in the MCP Store.

## Environment variables
- `BUILDBETTER_API_KEY` (required)
