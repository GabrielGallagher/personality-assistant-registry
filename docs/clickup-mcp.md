# ClickUp MCP setup

Use this MCP to read and update tasks in ClickUp.

## What you need
- A ClickUp account
- A personal API token

## Steps
1. In ClickUp, go to Settings -> Apps.
2. Generate a personal API token.
3. Set `CLICKUP_API_TOKEN` in the MCP Store.

## Environment variables
- `CLICKUP_API_TOKEN` (required)
