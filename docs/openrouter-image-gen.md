# OpenRouter Image Gen

Built-in OpenRouter image generation connector (core).

## Availability
- Bundled with the desktop app (no install)
- Enable the connector for your agent in Agent Settings -> Connectors

## Required Environment Variables
- `OPENROUTER_API_KEY` (required): OpenRouter API key (starts with `sk-or-`).

## Notes
- This connector is core and cannot be uninstalled.
- Use the core update flow to apply registry updates when available.
