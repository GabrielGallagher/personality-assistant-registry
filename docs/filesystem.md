# Filesystem MCP

Built-in filesystem access for approved local files and folders.

## Availability
- Bundled with the desktop app (no install)
- Enable per agent in Agent Settings -> Connectors

## Permissions
- Access is limited to approved workspace and manifest roots.
- Write access is constrained to configured write roots.

## Notes
- This connector is core and cannot be uninstalled.
- Use the agent filesystem toggle to enable or disable per agent.
