# Context7 MCP setup

Context7 provides up-to-date library docs and code examples on demand.

## What you need
- Optional Context7 API key (recommended for higher rate limits)

## Steps
1. (Optional) Get an API key at https://context7.com/dashboard
2. Set `CONTEXT7_API_KEY` if you have one.
3. Enable the MCP server in the MCP Store.

## Environment variables
- `CONTEXT7_API_KEY` (optional)
